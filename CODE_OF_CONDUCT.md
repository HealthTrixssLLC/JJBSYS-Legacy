# Code of Conduct

This project is a technical preservation archive. We want discussion to be respectful, careful, and useful to future readers.

## Our Expectations

- Be respectful in technical discussions.
- Focus on constructive review, documentation, and preservation.
- Treat historical code and fellow contributors with care.
- Share expertise generously, especially when explaining legacy CICS or mainframe concepts.

## Unacceptable Behavior

- Harassment or personal attacks.
- Dismissive or hostile conduct.
- Posting secrets, private operational details, or sensitive findings in public.

## Project Tone

This repository exists to preserve technical history and support learning. Contributions and reviews should reflect that purpose.

For conduct concerns, contact the maintainers privately.
