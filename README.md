# Legacy CICS Mainframe Archive

This repository preserves a legacy CICS/mainframe codebase as a historical software archive.

The project includes CICS, assembler, COBOL, JCL, EXCI bridge, TCP/IP, SMTP, web interface, operational automation, and system utility components.

## Purpose

The purpose of this repository is to preserve, document, and share the code for historical and educational use.

## Open Source Readiness

The repository has been sanitized for public release by replacing obvious secrets, user IDs, mailboxes, hostnames, keyrings, and other site-specific integration values with generic placeholders.

Some environment-specific values are still intentionally represented as installation examples inside legacy JCL and CICS assets. Review and localize those values before compiling or deploying in any environment.

See [OPEN_SOURCE_RELEASE.md](/Volumes/ExternalData/Projects/JJBSYS-Legacy/OPEN_SOURCE_RELEASE.md) for the release checklist, placeholder conventions, and remaining review guidance.
