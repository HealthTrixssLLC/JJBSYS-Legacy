# JJBSYS Code

This repository preserves the JJBSYS legacy mainframe/CICS systems codebase.

The project appears to include CICS, assembler, COBOL, JCL, EXCI bridge, TCP/IP, SMTP, web interface, operational automation, and system utility components.

This repository is currently maintained as a private legacy preservation project.

## Purpose

The purpose of this repository is to preserve, document, and evaluate the code for historical, educational, commercial, or open-source use.

## Important Notice

This code should be reviewed for ownership, security, environment-specific configuration, credentials, hostnames, IP addresses, dataset names, and customer-specific information before any public release.
