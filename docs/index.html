What this project is

JJBSYS-Legacy is a historical CICS and mainframe archive that preserves legacy systems programming work as a documentation, research, and modernization reference.

It captures practical patterns in CICS operations, batch integration, EXCI bridging, TCP/IP and SMTP communications, web support, user exits, terminal support, and operational automation.
