What this project is

JJBSYS-Legacy is a historical CICS and mainframe archive that preserves legacy systems programming work as a documentation, research, and modernization reference.

It captures practical patterns in CICS operations, batch integration, EXCI bridging, TCP/IP and SMTP communications, web support, user exits, terminal support, and operational automation.

Safety Notice

This repository is not production-ready software. It should not be compiled, installed, or executed without expert CICS, z/OS, security, compatibility, and local operational review.

Why it matters

Historical record

The repository preserves a real-world body of systems programming work in a form that future engineers can study.

Technical learning

It helps explain how CICS environments were extended, integrated, monitored, and operated.

Modernization reference

It provides concrete patterns that can be mapped to modern integration and operations approaches.
