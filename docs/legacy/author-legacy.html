JJBSYS Legacy CICS Mainframe Archive preserves a body of mainframe systems programming work originally authored by James J. Baker.

The project is intended to honor his work and help future engineers learn from practical CICS systems programming patterns preserved in the repository.

Purpose of this legacy page

This page exists to keep the repository grounded in its preservation purpose. The archive is not only a collection of source members. It is also a record of real technical work, practical systems problem-solving, and long-term engineering craft in a mainframe setting.
